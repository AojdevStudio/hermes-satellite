- The dispatch row is evidence only that a subagent was launched with goals/delegation_id; it is not evidence that the delegated review/task succeeded.
- Completion requires a later durable result row or assistant/user message that contains the subagent result and can be linked by delegation_id, task id, or explicit quoted review context.
- If no linkable completion row exists in T2, verifier must keep delegated claims PENDING/UNKNOWN and may use later assistant PASS prose only as assistant_assertion with low cap.
- User-pasted review text in reply context is external conversational evidence; preserve message id/source and do not merge it with the original delegate_task unless link evidence is explicit.
